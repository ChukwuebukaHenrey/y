‚öôÔ∏è Animation Properties Reference

/* Complete Animation Syntax */ .element { animation: name duration timing-function delay iteration-count direction fill-mode play-state; } /* Individual Properties */ animation-name: slideFade; /* Keyframe name */ animation-duration: 2s; /* How long animation takes */ animation-timing-function: ease-in-out; /* Speed curve */ animation-delay: 0.5s; /* Delay before start */ animation-iteration-count: infinite; /* How many times to repeat */ animation-direction: alternate; /* Direction of animation */ animation-fill-mode: forwards; /* What styles to apply when not running */ animation-play-state: paused; /* Whether animation is running or paused */ /* Timing Functions */ linear /* Constant speed */ ease /* Slow start, fast middle, slow end (default) */ ease-in /* Slow start */ ease-out /* Slow end */ ease-in-out /* Slow start and end */ cubic-bezier(0.25, 0.1, 0.25, 1) /* Custom timing curve */ /* Transform Functions */ translateX(50px) /* Move horizontally */ translateY(20px) /* Move vertically */ translate(10px, 20px) /* Move both directions */ rotate(45deg) /* Rotate */ scale(1.5) /* Scale uniformly */ scaleX(2) /* Scale horizontally */ scaleY(0.5) /* Scale vertically */ skewX(15deg) /* Skew horizontally */ skewY(15deg) /* Skew vertically */ /* Multiple transforms */ transform: translateX(50px) rotate(45deg) scale(1.2);

üí° Best Practices & Tips

/* Performance Tips */ 1. Use transform and opacity for animations (they don't trigger layout/paint) 2. Add will-change property for complex animations: will-change: transform, opacity; /* Smooth animations */ 3. Use transform3d() to enable hardware acceleration: transform: translate3d(0, 0, 0); /* Accessibility */ 4. Respect user preferences: @media (prefers-reduced-motion: reduce) { .animated-element { animation: none; } } /* Cross-browser compatibility */ 5. Add vendor prefixes for older browsers: -webkit-animation: slideIn 1s ease-out; -moz-animation: slideIn 1s ease-out; animation: slideIn 1s ease-out; /* Debugging animations */ 6. Use animation-play-state: paused; to pause animations for debugging 7. Use Chrome DevTools Animation Inspector for fine-tuning /* Common Patterns */ 8. Chain animations using animation-delay 9. Use CSS variables for dynamic animations: animation-duration: var(--animation-speed, 1s); 10. Combine with CSS Grid/Flexbox for responsive animated layouts
